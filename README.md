# Microsoft Entra IAM Enterprise Lab  
Tenant: RLSecurityConsultant.onmicrosoft.com  

This lab demonstrates enterprise-grade Identity & Access Management architecture using Microsoft Entra ID.

---

# Lab Phases

## Phase 1 – Identity Architecture & RBAC Design
User provisioning, structured attributes, group-based RBAC, role assignment via security groups.

Location:
Phase1-Identity-Architecture/

---

## Phase 2 – Conditional Access Architecture
Organization-wide MFA enforcement with geo-based logic and hardened admin policy.

Location:
Phase2-Conditional-Access/

---

## Phase 3 – Privileged Identity Management (PIM)
Just-In-Time Global Administrator with approval workflow and audit validation.

Location:
Phase3-PIM/

# Security Principles Demonstrated

• Least Privilege  
• Group-based RBAC  
• Conditional Access enforcement  
• Just-In-Time elevation  
• MFA enforcement  
• Audit traceability  
• Zero Trust alignment
